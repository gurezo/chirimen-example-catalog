# CHIRIMEN Example Catalog

> **アーカイブ済み（読み取り専用）**  
> Platform 別 Example の正本は [chirimen-device-dashboard](https://github.com/gurezo/chirimen-device-dashboard) に移行しました。  
> 最新の Platform 別 Example は [公開サイト](https://chirimen-device-dashboard.web.app/) で確認してください。

CHIRIMEN Example Catalog は、CHIRIMEN 関連リポジトリに散在している Example を、デバイス別・プラットフォーム別に整理するためのドキュメントです。

## このドキュメントで分かること

- デバイスごとの platform 別サンプルコード
- 現行サンプル（primary）と旧サンプル（legacy / archive）の違い
- upstream リポジトリとの対応関係
- 推奨 Example と実機確認状況
- [chirimen-device-dashboard](https://github.com/gurezo/chirimen-device-dashboard) への参照

## このドキュメントで扱わないこと

以下は [chirimen-device-dashboard](https://github.com/gurezo/chirimen-device-dashboard) を正とします。

- 商品リンク・商品画像
- 回路図・データシート
- デバイス説明・カテゴリ・タグ

## 前提

このドキュメントは、`examples/devices/**/metadata.md` と `catalog/*.json` を元に生成・更新されます。

## 注意

本ドキュメントサイトはアーカイブ済みリポジトリの参照用です。  
Platform 別 Example の更新は [chirimen-device-dashboard](https://github.com/gurezo/chirimen-device-dashboard) の `data/platform-examples/` で行ってください。
